저작권
문제의 저작권은 문제를 만든 사람에게 있습니다.
소스 코드의 저작권은 소스 코드를 작성한 사람에게 있습니다.
문제의 본문을 블로그에 업로드하는 것 보다는 링크로 추가해주세요.
BOJ를 활용한 인터넷 방송은 자유롭게 할 수 있습니다.

따라서 모든 소스 코드에는 문제 번호, 문제 이름, 해당 문제 링크를 작성합니다.
프로그래밍 언어는 Python입니다.
제 백준 아이디는 kjk9704입니다.

아, 그리고 백준 사이트 목적은 "알고리즘 공부"이기 때문에
공부를 위해 제 소스 코드 보고 자신만의 스타일로 변형하는 것은 괜찮고
solved.ac 기준 골드 이하의 소스 코드는 코딩 테스트 연습하는 분들이 대부분이기 때문에
골드 이하의 소스 코드는 복붙 제출해도 상관 없긴 합니다.
하지만 solved.ac 기준 플레티넘 이상의 소스 코드는 ctrl C + ctrl V로 복붙 치팅은 하지 말아주세요.
플레티넘 이상의 소스 코드는 코딩 테스트 아닌 PS 목적으로 하기 때문에 반드시 자신만의 스타일로 변형해주세요.

[2025-08-06 추가]
이번에 solved.ac에 존재한 에센셜 문제 풀이 소스 코드를 공유합니다.
solved.ac는 백준과 연동된 사이트이며 문제들의 난이도를 티어로 표시하는 사이트입니다.
solved.ac 기준 class 5까지는 코딩테스트 공부하는데 도움이 되는 문제들로 이루어져 있고
class 6부터는 알고리즘 대회 문제들이 많이 등장합니다.

class 5까지는 코딩테스트 공부하는 분들을 위해 제 소스 코드를 코드 분석해서 공부해도 됩니다.
어차피 취업 목적으로 하기 때문에 이런 것까지 복붙 막는 것은 아니라고 판단합니다.
(쉽게 말해 class 5 이하까지는 티어 올리는 목적 아닌 것으로 판단)
하지만 class 6 이상은 절대로 치팅하지 마세요. 제 소스 코드는 그저 참고용이지 티어 올리는 용도가 아닙니다.
그럼에도 class 6 이상 소스 코드들을 공유한 이유는 데이터 분석 혹은 알고리즘 연구하는 분들을 위해 공유합니다.
그러므로 자신만의 스타일로 소스 코드를 변형해주세요.
